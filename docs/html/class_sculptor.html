This is the Sculptor class' constructor method.

This constructor method dynamically allocates a 3D matrix defined by the given parameters, which the class' methods utilize to draw on. It also saves the dimensions of the matrix and sets the drawing color as (r=0, g=0, b=0, a=0).

Parameters
_nxThe x dimension size.
_nyThe y dimension size.
_nzThe z dimension size.
